# Contributing

1. Fork the repo
2. Clone your fork
3. Install Rust: https://rustup.rs
4. Build: `cargo build`
5. Test: `cargo test`

Comment on an issue to claim it. Run `cargo fmt` and `cargo clippy` before submitting a PR.

## Labels

- `good first issue` — no prior Soroban knowledge needed
- `help wanted` — moderate complexity
- `core` — requires Soroban/XDR knowledge
